# 寻域 · DomainGo

> **Explore the present. Discover the city.**

**寻域（DomainGo）** 是一款基于定位的城市探索与导览应用，围绕「城市认知」「本地经验」「出行引导」三大核心场景，帮助用户在当下所在城市中发现值得了解、值得体验、值得记录的内容。

------

## 📱 项目简介

当你来到一座城市，真正有价值的信息往往不是“必去榜单”，而是：

- 这座城市有哪些独特的地名与文化背景？
- 本地人和游客都在如何评价这里？
- 有没有适合当前时间、当前地点的游玩路线？
- 我来过哪些地方，又留下些什么记忆？

**寻域** 正是围绕这些问题而设计。

应用以用户实时定位为基础，整合城市地名信息、用户评论与游玩攻略，为用户提供「探索 + 社交 + 导览」的一体化城市体验。

------

## ✨ 核心功能

### 🌍 探索（Explore）

- 基于定位自动识别当前城市
- 展示城市地名信息与历史背景（数据来源：**博雅地名网**）
- 帮助用户快速了解城市概况与区域特色

------

### 💬 社交（Social）

- 发布针对**当前城市**的评论、心得与攻略
- 按城市维度查看他人分享的内容
- 强调真实体验与在地视角，而非单纯“打卡”
- 寻求你的专属“旅游搭子”！

------

### 🧭 导览（Guide）

- 提供推荐的城市游玩路线
- 结合地理位置，辅助用户规划行程
- 面向轻度旅行与城市漫游场景设计

------

### 🗺️ 我的足迹（Footprints）

- 自动记录用户到访过的城市或地点
- 构建个人的城市探索轨迹
- 让出行不只是一次性体验，而是可回顾的记忆

------

## 🧠 产品理念

- **以“当下所在城市”为中心**，而非泛化的旅行榜单
- 重视真实用户体验与在地内容
- 轻量但有深度，适合随走随看、随用随停

**寻域** 更像是一位安静的城市向导，而不是强行安排路线的“攻略机器”。
